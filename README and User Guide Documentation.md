# TaskCLI – Java Task Management System Documentation

## Project Name
TaskCLI (Java Version)

## Description
TaskCLI is a lightweight Java-based command-line task management system that allows users to create, view, and complete tasks. Each task can be assigned a priority level and a due date to assist with basic task organisation and tracking.

---

# 1. README (Prompt 1 Output)

## Features
- Add new tasks
- View all existing tasks
- Mark tasks as completed
- Assign priority levels (LOW, MEDIUM, HIGH)
- Set due dates for tasks

## Technologies Used
- Java
- Arrays / ArrayList
- Object-Oriented Programming (OOP)
